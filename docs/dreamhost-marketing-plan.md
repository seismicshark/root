# DreamHost marketing plan

## Hosting target
Existing DreamHost shared unlimited hosts static marketing properties for product acquisition while platform ownership remains under `seismicshark.com`.

## Marketing subtree
- `public_html/getcallintakepro.com/`
- `public_html/morejobsfromcalls.com/`
- `public_html/turncallsintoleads.com/`
- `public_html/callstobookings.com/`

## Page intent
Each site should remain simple:
- one clear promise
- one primary CTA
- one path into `callintakepro.com`
- no app runtime dependency
- static assets only

## Ownership rule
- SeismicShark owns registrar, DNS, vendor accounts, and recovery identities
- product domains remain brand and acquisition surfaces
- no product domain should be used as the root ownership email for vendors

## Deployment pattern
Author pages in repo, export/copy to DreamHost public_html for each domain.
